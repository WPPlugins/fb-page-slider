Add Facebook Page Slider to your website to enhance the experience of your website’s visitors. Facebook Page Slider provides immediate access to your Facebook pages, Facebook posts etc.

== Description ==

Facebook Page Slider is compatible with all Facebook pages and Wordpress sites. After downloading and installing Facebook Page Slider your website visitors will gain access to different aspects of your Facebook pages.

== Installation ==

This section describes how to install the plugin and get it working.

1. Install the wordpress plugin first

You can install it via Wordpress Installer or can download the file first then Upload it on /wp-content/plugins/ directory

2. Activate the plugin

3. You will see facebook settings from right side of your wordpress menu.

Settings Configuration:

Facebook Page URL : Put your Facebook page url here. Example: http://facebook.com/facebook

Alignment: You can position your facebook box either on left or right using this option.

Margin Top - Height & Width : Their you have to Put Numeric Number as per your need.
